# 今天只做一件事

这是一个给第一次玩 Vibe Coding 的人准备的极简 Demo。

## 最初的 Vibe

> 我每天列很多待办，最后一件都没完成。我想要一个页面，每天只记一件最重要的事。

## 验收标准

- 一次只保存一件事。
- 刷新页面后内容不丢失。
- 可以选择当下心情。
- 完成后有明确反馈。
- 手机上也能舒服使用。

## 怎么玩

1. 先说出一个真实的小问题，不急着选框架。
2. 让 AI 把想法收敛成 3～5 条验收标准。
3. 一次只实现一个小步骤，每一步都能看到变化。
4. 发现不对时描述现象和感受，不要只说“优化一下”。
5. 能用以后再补文档和细节。
