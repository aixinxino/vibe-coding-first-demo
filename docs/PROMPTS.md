# 从灵感到上线的五轮对话

## 1. 先不写代码
> 我想做一个每天只记一件事的小网页。请先把这句话变成 5 条能验收的要求，不要扩展账号、服务器和社交功能。

## 2. 先看到东西
> 用原生 HTML 做第一版，只包含标题、输入框和保存按钮。先不加框架，也不要过度设计。

## 3. 描述感受
> 现在太像普通表单。我想要安静、轻松、没有压力的感觉，手机上按钮要容易点。

## 4. 补真正有用的行为
> 用 localStorage 保存今天的任务。刷新不能丢失，完成后可以清空，空内容不能保存。

## 5. 最后检查和记录
> 检查键盘操作、手机布局和空状态，再在 README 记录最初想法、验收标准和使用方法。

下一步只选一个方向：历史记录、分享卡片或新的视觉主题。不要一次把所有想法都塞进去。
